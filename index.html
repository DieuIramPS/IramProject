Branches Git

Une branche Git est une ligne de développement indépendante qui permet de travailler sur des fonctionnalités sans affecter la branche principale.

Commandes principales :

  • git branch - Lister les branches
  • git branch nom-branche - Créer une nouvelle branche
  • git checkout nom-branche - Basculer vers une branche
  • git merge nom-branche - Fusionner une branche

Cette branche option2 contient des modifications spécifiques pour tester les fonctionnalités de Git et GitHub.
